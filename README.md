# Temat: JustGrades - Student Grading System
The project is a team exercise primarily aimed at acquiring database design skills. The result of the project is to be a functional system consisting of a database along with a simple application that utilizes this database for both operational activities and analytical reporting.

## Environment Setup
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash

install node.js

nvm install 22

npm install -g pnpm@latest-10

nvm use 22

npm install @mui/material @emotion/react @emotion/styled

npm install chart.js react-chartjs-2

## Running the application
* just-grades-rest (spring):

 clear && ./mvnw spring-boot:run


* just-grades-ui (react):

 pnpm dev

WebSite:

 lockalhost:3000

 swagger dokumentation: http://localhost:8080/swagger-ui/index.html#/   


## Project Implementation:
* baza danych typu relacyjnego: Oracle
* backend: Java + Spring 
* frontend: React
